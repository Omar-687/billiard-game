BILLIARD GAME 

HOW TO RUN THE GAME 
Go to dist folder and then billiard folder. After that, you can see 
that there is billiard file (application). Click on it. Terminal and game window will open. (You can close the terminal, it is not important for the game)

WHAT TO DO DURING GAME
To move the red ball, you need to hold left click behind the ball, so
you can make resulting movement stronger/weaker depending on the distance
of line. Then you shoot the ball by releasing left click.
